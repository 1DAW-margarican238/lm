Explicación de las propiedades usadas externas

1. height: 100vh - Pantalla completa

La propiedad "height: 100vh" hace que el elemento ocupe toda la altura de la ventana del navegador.

2. center/cover - Imagen de fondo

La propiedad background con center/cover se usa para que la imagen de fondo se muestre centrada y cubra todo el contenedor sin distorsionarse.

3. Explicación del Script - Slideshow

Este script es para crear un carrusel de imágenes. Permite cambiar las imágenes que se muestran al hacer clic en los botones. He tomado este código de Internet, específicamente de W3Schools.

4. Audio - Agregar audio

Se pueden agregar archivos de audio con la etiqueta <audio>. He incluido la opción controls para permitir que el usuario controle la reproducción.

5. radial-gradient - Gradiente radial con imagen de fondo

El código radial-gradient(circle, rgba(0, 0, 0, 0.5) 0%, rgba(0, 0, 0, 0) 90%), url('imagenes/sevilla/fondosevilla.jpg') center/cover; aplica dos efectos visuales: radial-gradient crea un gradiente circular que oscurece el centro de la imagen, y url('imagenes/sevilla/fondosevilla.jpg') center/cover define la imagen de fondo, para que se centre y cubra todo el espacio sin distorsionarse.
