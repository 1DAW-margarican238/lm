Explicación de las propiedades usadas externas

1. height: 100vh - Pantalla completa

La propiedad "height: 100vh" hace que el elemento ocupe toda la altura de la ventana del navegador. "vh" es una unidad de medida relativa al 1% de la altura de la ventana.

2. center/cover - Imagen de fondo

Usamos la propiedad background con center/cover para asegurar que la imagen de fondo se muestre centrada y cubra todo el contenedor sin distorsionarse.

3. Explicación del Script - Slideshow

Este es el script para un carrusel de imágenes. Definimos un array llamado slideIndexes para llevar un seguimiento del índice actual de las imágenes en cada grupo de slides. Hay una función plusDivs que se encarga de aumentar o disminuir el índice de la imagen mostrada cuando se hace clic en los botones del carrusel. La función showDivs se usa para mostrar o esconder las imágenes dependiendo de su índice. El script también inicializa el carrusel mostrando la primera imagen al cargar la página.

4.

Se pueden agregar archivos de audio con la etiqueta <audio>. Esto permite reproducir sonidos en la página. Se puede incluir la opción controls para permitir que el usuario controle la reproducción.
